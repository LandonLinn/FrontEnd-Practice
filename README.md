## ⚡ Ground Rules for 30-Min Sessions

| Time       | Activity                                                           |
| ---------- | ------------------------------------------------------------------ |
| 0–5 min    | Review yesterday — open your code, read it, recall what you built  |
| 5–20 min   | Learn the concept (video at 1.5x OR read docs — pick one)          |
| 20–30 min  | Build the mini-component. **No copy-paste. Type every line.**      |
| End of day | Commit to GitHub with a descriptive message. **Every single day.** |
